What Is This App?

The Constellation Generator creates a starfield based on a single word or prhase called a seed. The same seed always produces the same star pattern. Users can pan, zoom, draw constellation lines, and click stars to reveal their myths.

1. Seeding the Sky

A seed is any word a user types, like Orion or Andromeda. The app converts each character in the seed into a number, combines the numbers, and uses that value to guide the random number generator. This ensures the sky is repeatable:the same seed always produces the same star pattern.

2. Deterministic Randomness

The app uses a simple mathematical function to create random-looking numbers from the seed. These numbers detemrine where each star goes. Because the math always starts from the same seed value, the results look random but are actually predictable and repeatable.

3. Starfield

Each star gets an X and Y position based on the assigned random values. The canvas then draws each star as a tiny glowing circle. Panning and zooming simply moves or scales the view. You're looking at the same starfield, just closer or farther away.

4. Constellation Lines

The app connects nearby stars based on two controls:

  • Max Distance: how close two stars must be to connect
  • Max Neighbors: how many connections each star can have

Changing these settings redraws the connections, creating new constellation shapes from the same stars.

5. Star Myths

Clicking a star sends its name to the server, which returns a short, story-like myth about that star. The myths are generated from themed word lists to feel ancient, mysterious, and atomospheric, like real constellation legends.
